# Asistente de Código con IA

Asistente de programación inteligente desarrollado en Python que responde dudas de código con ejemplos claros.

## Tecnologías usadas
- Python
- Groq API (LLaMA 3.3)
- Librería groq

## Funcionalidades
- Chat interactivo con memoria conversacional
- Respuestas con ejemplos de código
- Guarda el historial de conversación en un archivo .txt

## Cómo usarlo
1. Clona el repositorio
2. Instala las dependencias: `pip install groq`
3. Añade tu API key de Groq en el código
4. Ejecuta: `python "introducion con ia .py"`
